# useful

Полезные утилиты и сниппеты, которыми можно делиться публично. Репозиторий организован по стеку и тематике, чтобы находить инструменты было легко.

## Структура

- `python/` — скрипты и утилиты на Python.
  - каждая подпапка — отдельный инструмент: свой `README.md`, зависимости и примеры запуска;
  - рабочие переменные хранятся в `.env`, рядом лежит `.env.example`.
- `javascript/`, `sql/`, `notebooks/` — при необходимости добавляем аналогичные каталоги.
- `docs/` — общие заметки, инструкции, шаблоны.

Добавляя новую «полезняшку», создавайте подпапку с понятным названием, кладите туда:

1. исходный код;
2. `README.md` с кратким описанием, требованиями и примером запуска;
3. `requirements.txt` / `package.json` / другой файл зависимостей;
4. `.env.example`, если нужны секреты.

## Wordstat: топ запросов

Первый инструмент — CLI-скрипт для получения топовых запросов из API Яндекс.Wordstat.

```bash
cd python/wordstat
cp .env.example .env  # вставьте WORDSTAT_API_KEY
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
python wordstat_top_requests.py "продвижение сайта" "контекстная реклама"
```

Скрипт вернёт JSON по каждой фразе. Токен берётся из `WORDSTAT_API_KEY` в `.env` или из переменных окружения. Файл `.env` не коммитим — он в `.gitignore`.
Документация по получению доступа к API: https://yandex.ru/support2/wordstat/ru/content/api-wordstat
